# Python Problem Solving 🐍

A collection of Python projects focused on problem solving, probability, simulations, algorithms, and mathematical thinking.

The goal of this repository is to improve programming skills through practical exercises and computational experiments rather than simple syntax practice.

---

## Projects

### 1. Birthday Paradox

Estimate the probability that at least two people in a group share the same birthday using a Monte Carlo simulation.

**Concepts:**

* Probability
* Sets
* Random Sampling
* Monte Carlo Simulation

---

### 2. Monty Hall Simulation

Simulate the famous Monty Hall problem and compare the outcomes of switching versus staying with the initial choice.

**Concepts:**

* Conditional Probability
* Decision Making
* Monte Carlo Simulation
* Random Events

---

### 3. Buffon's Needle

Estimate the value of π through a geometric probability experiment involving randomly dropped needles.

**Concepts:**

* Geometric Probability
* Trigonometry
* Numerical Approximation
* Monte Carlo Simulation
